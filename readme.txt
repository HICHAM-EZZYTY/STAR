Nous voulons créer un widget HTML qui affichera la note des marques récupérée depuis une api JSON. Ce widget devra s'intégrer sur des sites web avec du Javascript d'une façon asynchrone. Ce widget devra ressembler au fonctionnement du Like button de Facebook (https://developers.facebook.com/docs/plugins/like-button/). Le widget devra être configurable à travers un node HTML avec les attributs "data" (Ex: <div id="MyWidget" data-brand="1234"></div>) et qui sera placé dans le code source de la page web qui affichera le widget.

Le Widget récupérera la note depuis une api JSON mais pour le test on se limitera à utiliser le fichier "data.json" comme source de données.

Le Widget devra être intégrer sur une page web directement sur son code et NON en frame !

Possible d'utiliser ReactJS ou VueJS

La template du widget et disponible dans le dossier "template"
